# SPACO - Service Providers All Connected Online (Flutter App)

SPACO (Service Providers All Connected Online) is a mobile application built with Flutter, designed to connect users with local service providers. This platform enables users to easily hire professionals for services such as plumbing, air-conditioning maintenance, and house cleaning.

**Features**

- **User Module**
  - Register, login, and manage user profiles.
    ![DPT5101_FYP_FINAL_REPORT_TEMPLATE_WEB_MOBILE_TRI2020](https://github.com/user-attachments/assets/e6310f97-486a-4ac8-a5c8-35327d0c545e)

  - Browse a marketplace of service providers.
    
    ![service details](https://github.com/user-attachments/assets/0440d500-6ffd-49a1-9557-6937b707c041)  
    
  - Add services to cart, book appointments, and make payments securely.
    ![mycart-book-payment](https://github.com/user-attachments/assets/f198082a-46dd-417c-93d0-602eaf420f41)
  
- **Service Provider Module**
  - Service providers can register, sign up, and create a profile.
    ![service provider login-register-profile](https://github.com/user-attachments/assets/3f2aa258-fa8e-41fd-9ac2-0a4d37c2aee3)

  - Service providers panel, where they can manage their services and view jobs.
    ![service provider home-services-job](https://github.com/user-attachments/assets/214a31de-a9ec-4359-b940-b49aae238b71)

  - View job requests and job status (Pending / Accepted / Ongoing).
    ![service provider my jobs status](https://github.com/user-attachments/assets/4cf71c1a-5191-4921-a8d9-1add383fa6ab)

  - Integration of QR code scanning for job completion verification.
    ![service provider my jobs -customer confirm-qr](https://github.com/user-attachments/assets/55cf50d2-e3a4-4ea5-8cc4-c0a7a7fb3bff)

- **Admin Module**
  - Admin interface to verify and manage service providers.
    ![admin home-verify](https://github.com/user-attachments/assets/b3257298-23dc-4bcb-876c-5ee85cf36bb1)

  - View and manage services, users, and service providers.
    ![admin view all services-users-sp](https://github.com/user-attachments/assets/60f2d12c-f0f3-4644-b16c-9bce3adc0a60)

**Tools Used**
